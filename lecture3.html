Nested Lists

Definition and Usage

A nested list means placing a list inside another list. Both <ul> (unordered list) and <ol> (ordered list) can be nested.

Types of Nested Lists:

  • Nested Unordered List (ul inside ul)
  • Nested Ordered List (ol inside ol)
  • Mixed Nested List (ol inside ul or ul inside ol)

Code Screenshot

Example 1: Nested Unordered List

Code:

<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Mango</li>
      <li>Banana</li>
    </ul>
  </li>
  <li>Vegetables
    <ul>
      <li>Carrot</li>
      <li>Potato</li>
      <li>Spinach</li>
    </ul>
  </li>
</ul>

Output:

  • Fruits
    • Apple
    • Mango
    • Banana
  • Vegetables
    • Carrot
    • Potato
    • Spinach

Example 2: Nested Ordered List

Code:

<ol>
  <li>Frontend
    <ol>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ol>
  </li>
  <li>Backend
    <ol>
      <li>Node.js</li>
      <li>Python</li>
      <li>Java</li>
    </ol>
  </li>
</ol>

Output:

  1. Frontend
    1. HTML
    2. CSS
    3. JavaScript
  2. Backend
    1. Node.js
    2. Python
    3. Java

Example 3: Mixed Nested List

Code:

<ol>
  <li>Web Development
    <ul>
      <li>Frontend</li>
      <li>Backend</li>
      <li>Database</li>
    </ul>
  </li>
  <li>Data Science
    <ul>
      <li>Python</li>
      <li>Machine Learning</li>
      <li>AI</li>
    </ul>
  </li>
</ol>

Output:

  1. Web Development
    • Frontend
    • Backend
    • Database
  2. Data Science
    • Python
    • Machine Learning
    • AI
